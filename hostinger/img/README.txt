GÖRSELLER — buraya 5 dosya koyun (PNG önerilir). İsimler AYNEN şöyle olmalı:

  gorsel1.png  -> EP01 forklift          (1 numaralı görsel - TÜMOSAN kırmızı 4 teker)
  gorsel2.png  -> EP02 / 3 teker forklift (2 numaralı görsel - CPD16)
  gorsel3.png  -> EP02 / 4 teker forklift (3 numaralı görsel - orsismak)
  gorsel5.png  -> Batarya                 (5 numaralı görsel - siyah akü kutusu)

Panel bu dosyaları otomatik gösterir. Dosya yoksa yerine 🚜 / 🔋 emoji çıkar (hata vermez).
Araç eklerken EP01/EP02 + 3/4 teker seçimine göre doğru görsel önizlemede belirir.

Yayına almak için deploy.ps1 çalıştırın; bu klasördeki dosyalar da /img/ altına yüklenir.
